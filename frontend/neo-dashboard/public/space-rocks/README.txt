Drop your JPEGs in this folder so the Login page can load them via absolute paths.

Expected filenames (used in src/pages/Login.jsx):
- asteroid.jpg
- comet.jpg
- meteor.jpg
- meteorite.jpg
- bolide.jpg

You can replace these with any images you like as long as you keep the same filenames,
or update the <img src="/space-rocks/..."> paths in Login.jsx.
